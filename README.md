# AI Data Analyst Agent

An AI-powered dashboard that analyzes CSV datasets using OpenAI and Pandas.
# AI Data Analyst Agent

## Features                             
- CSV file Upload
- Dataset Preview
- Missing Value Detection
- Natural Language Query Interface
- AI Insight Engine
- Data cleaning
- Line Charts
- Bar Charts
- Pie Charts
- Heatmaps

## Data Visualization
- Line Charts (Trend Analysis)
- Bar Charts (Comparisons)
- Pie Charts (Distribution Analysis)
- Correlation Heatmaps

## Data Processing
- Data Cleaning
- Statistical Analysis
- Missing Data Identification
- Numerical Feature Analysis

## Tech Stack 
- Python
- Flask
- Pandas
- Matplotlib
- Seaborn
- HTML/CSS/JS
- OpenAI API

## Setup
- git clone https://github.com/akshayareddy03/ai-data-analyst-agent.git
- pip install -r requirements.txt

## Run
python app.py
## 📖 How to Use

1. Upload a CSV dataset.
2. Preview the dataset.
3. View missing value statistics.
4. Generate visualizations automatically.
5. Ask questions about the dataset.
6. Receive AI-powered insights and recommendations.
